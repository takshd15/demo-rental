# NestRoom Website Requirements

## 1. Global Navigation

The website must include a navigation bar that appears on all pages.

**Requirements:**

- Display the NestRoom logo with "🏡 NestRoom" in the top-left
- Display menu items on the top-right:
  - Home
  - About Us
  - Contact
- Navigation should be consistent across the whole website
- Navigation should work well on both desktop and mobile

## 2. Home Page

The home page must introduce the platform and highlight featured listings.

**Requirements:**

- Include a Featured Rentals section
- Show title: "Featured Rentals"
- Show subtitle: "Explore our latest listings"
- Display a few rental cards
- Each rental card should include:
  - Image
  - Title
  - Price
  - Short description
- Include an About section preview
- Show short text such as:
  - "Helping you find a place to call home"
  - "We provide rental properties with a little touch, let us help you find your next nest with ease"
- Include a Learn More button linking to the full About page

## 3. Listings Page

The listings page must let users browse all available rooms.

**Requirements:**

- Show all rooms in a grid or list layout
- Each listing should display:
  - Photo
  - Title
  - Location
  - Price
  - Type of rental
  - Short description
- Users must be able to filter listings by:
  - Price
  - Location
  - Type (sublet or full rental)
- Users must be able to sort listings by:
  - Price
  - Location
  - Type

## 4. User Registration and Login

The system must allow users to create accounts and sign in.

**Requirements:**

- Users can register
- Users can log in
- Users can manage their own posted rooms and profile
- Authentication should be simple and secure

## 5. Room Posting Feature

Users must be able to create and publish room listings.

**Requirements:**

- Logged-in users can post a room
- Posting form must include:
  - Title
  - Description
  - Location
  - Price
  - Photos
  - Contact information
  - Rental type (sublet or full rental)
- Contact information may include:
  - Phone number
  - Social media links
- Users should be able to edit or delete their own listings

## 6. Reviews System

Users must be able to leave reviews for rooms they stayed in.

**Requirements:**

- Allow users to submit reviews on room listings
- Reviews should be attached to the correct room
- Reviews should help future renters make decisions
- Only appropriate logged-in users should be allowed to leave reviews

## 7. Roommate Finder

The platform should include a roommate-matching section.

**Requirements:**

- Users can create a roommate profile
- Profile should include:
  - Name
  - Age
  - Lifestyle preferences
- Users should be able to match with:
  - Rooms
  - Other users looking for roommates

## 8. About Page

The website must include a dedicated About page.

**Requirements:**

- Include a short description of the platform
- Explain the purpose of NestRoom
- Expand on the message of helping users find homes easily
- Be accessible through the navigation bar and Learn More button

## 9. Contact Section / Contact Page

The website must provide contact information.

**Requirements:**

- Include contact details users can use to reach NestRoom
- Contact section should be easy to find
- Could include:
  - Email
  - Facebook
  - Other social links if needed

## 10. Footer

The website must include a footer on all pages.

**Requirements:**

- Show copyright text: © 2026 NestRoom
- Include links to:
  - Privacy Policy
  - Terms
- Include contact info such as:
  - Facebook
  - Email

## Functional Requirements

These are the things the system must do.

- Display featured rentals on the home page
- Show all listings on a dedicated listings page
- Allow filtering and sorting of listings
- Allow users to register and log in
- Allow users to post room listings
- Allow users to upload photos
- Allow users to provide contact information
- Allow users to mark listings as sublet or full rental
- Allow users to leave reviews
- Allow users to create roommate profiles
- Allow users to match with rooms or roommates
- Allow navigation between all main pages

## Non-Functional Requirements

These are the quality requirements.

- The design must be simple and clean
- The website must be easy to navigate
- The website must be mobile-friendly
- The interface should reduce stress and confusion for users
- The platform should make it easy to:
  - Post rooms
  - Leave reviews
  - Find rooms
  - Find roommates

## Suggested Main Pages

For clarity, the website will likely need these pages:

- Home
- Listings
- Listing Details
- About Us
- Contact
- Login
- Register
- Post a Room
- Roommate Finder
- User Profile
